# AirbnbClone
A new mobile app development in my React Native journey.

## **Latest achievements:** <br />
1- Implemented multiple and different looking screens. <br />
2- Dealt with bottom & top tab bars. <br />
3- Explored nested screens navigations. <br />
4- Worked with google places autocomplete api. <br />
5- Applied beautiful UI styling using various components and vector icons. <br />
6- Explored the setup & Implementation of google maps. <br />
7- Coded advanced UI such as google maps markers and carousel flatlist. <br />
8- Animated google maps based on flatlist visible item & custom markers. <br />
9- Screen navigation with parameters. <br />
10- Explore and conect AWS services to the app backend. <br />
11- Implement User authentications ( SignUp, SignIn, Forget Password & Email verification). <br />
12- Upload and Retrieve data from AWS cloud. <br />
13- Apply filtering on retrieved data (based on Max no. of Guests & Selected Location).  <br /> <br />

## **Next Target:** <br />
Enhance the app functionality & add modern features.<br /><br />
